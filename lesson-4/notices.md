# Lesson 4

This lesson covers aggregate functions, grouping rows, filtering grouped data, and using subqueries in PostgreSQL.

## Topics covered

- using aggregate functions like `MIN` and `MAX`
- finding a value from a whole table
- using subqueries inside `WHERE`
- finding rows based on aggregate results
- grouping rows with `GROUP BY`
- filtering rows with `LIKE`
- counting rows in each group
- using `FILTER` with aggregate functions
